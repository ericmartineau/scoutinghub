Error page

Stylish and discrete, the error page looks carved in wood.

Use the following syntax to create error pages:

  
    <div class="error-container center">
      <h1 class="transparent">#Error code</h1>

      <div class="error-details">
        Error message
      </div>

      <div class="error-actions">
        <a href="#" class="btn btn-large btn-orange">
          <i class="icon-chevron-left"></i>
          &nbsp;
          Back button
        </a>
      </div>
    </div>

#404

The page you have requested does not exist at this location.
