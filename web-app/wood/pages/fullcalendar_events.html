Full calendar with draggable events

This is a cool feature from jQuery - FullCalendar.

The draggable events are extra and, of course, optional.

All you have to do in order to create the full calendar is create a div with id="calendar" and class="calendar".

The rest is done automatically in the JavaScript files.

If you want to create draggable events, use the following syntax:

    
    <div id="draggable-events" >
      <h5>Draggable events</h5>
      <div>
        <div class="event bg-red ui-draggable" style="position: relative; ">Event 1</div>
        <div class="event bg-red ui-draggable" style="position: relative; ">Event 2</div>
        <div class="event bg-red ui-draggable" style="position: relative; ">Event 3</div>
        ...
        <p>
          <label for="drop-remove">
            <div class="checker" id="uniform-drop-remove">
              <span>
                <input type="checkbox" id="drop-remove" style="opacity: 0; ">
              </span>
            </div>
            Remove event after drop
          </label>
        </p>
      </div>
    </div>
     

    
You can also remove an event from the list after dragging it over the calendar by checking "Remove
    after drop".
