Buttons

In order to style an element as a button use class="btn"

For various sizes, add one of the following classes:

btn-large btn-block btn-large btn-small btn-mini

You can also establish the color of the button by adding one of the following classes:

default btn-green btn-yellow btn-orange btn-red btn-blue btn-inverse

Button groups and Dropdowns

The button groups have the same syntax as original bootstrap. In order to change the color or the size of the buttons, add the classes mentioned in the Classic buttons documentation.

Button Button Button

Dropdown menus have exactly the same syntax as the ones in the original bootstrap. Here, as in button groups case, you can vary the colors and the sizes of the buttons.

Example:

    
      <div class="btn-group">
        <button class="btn btn-yellow dropdown-toggle" data-toggle="dropdown">Warning <span class="caret"></span></button>
        <ul class="dropdown-menu">
          <li><a href="#">Action</a></li>
          <li><a href="#">Another action</a></li>
          <li><a href="#">Something else here</a></li>
          <li class="divider"></li>
          <li><a href="#">Separated link</a></li>
        </ul>
      </div>

You can use class dropdown-left on a button toggle to make it open in the left side of the toggle (right aligned)

Square buttons

These big buttons are styled using class="button square" or class="button rectangular"

These buttons can be used as quick links and are enhanced with icons to be more suggestive.

The code for the example above:

    
      <button class="button square" href="#">
        <i class="icon-file icon-large"></i>
        <span class="button-text">New Article</span>
      </button>

You can use class dropdown-left on a button toggle to make it open in the left side of the toggle (right aligned)
