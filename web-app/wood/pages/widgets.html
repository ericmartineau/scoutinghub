Modal dialogs

This is the code behind the modal dialog. Basically you have to add "modal hide fade" classes to your div element.

    
      <div class="modal hide fade" id="myModal">
        <div class="modal-header">
          <button type="button" class="close" data-dismiss="modal" aria-hidden="true">&times;</button>
          <h3>Modal header</h3>
        </div>
        <div class="modal-body">
          <p>This is the modal dialog.</p>
        </div>
        <div class="modal-footer" data-dismiss="modal">
          <a href="#" class="btn btn-green">OK</a>
        </div>
      </div>

    </div>

Labels and badges

To create labels add label class to your element.

In addition to that, you can choose the color of your label using one of the following classes:

label-yellow label-red label-green label-blue label-orange label-inverse

To create badges add badge class to your element.

As in labels' case, change the color of your badge using one of the classes: badge-yellow, badge-red, badge-green, badge-blue, badge-orange, badge-inverse

Progress bars

In order to create progress bars, use the same syntax that bootstrap uses.


   <div class="progress">
      <div class="bar" style="width: 70%"></div>
    </div>

If you want a striped bar, add the class progress-striped to the element.

If you want an animated bar, add the classes progress-striped active to the element.

Sliders

The sliders are created using JQuery Sliders, and customized in order to suit the theme palette and style.


   <div class="slider slider-red" id="slider-range-min"></div>

In order to change the color of the slider, add one of the following classes: slider-red, slider-orange, slider-yellow, slider-green, slider-blue.

Tabbed content

Use the syntax below for creating tabbed content. 


      <div class="tabbable" style="margin-bottom: 18px;">
        <div class="tab-header">
            <span class="pull-right">
              <span class="options">
                <div class="btn-group">
                  <a class="dropdown-toggle" data-toggle="dropdown"><i class="icon-cog"></i></a>
                  <ul class="dropdown-menu black-box-dropdown dropdown-left">
                    ...
                  </ul>
                </div>
              </span>
            </span>
        </div>

        <ul class="nav nav-tabs">
          <li class=""><a href="#tab1" data-toggle="tab">Section 1</a></li>
          <li class=""><a href="#tab2" data-toggle="tab">Section 2</a></li>
          <li class="active"><a href="#tab3" data-toggle="tab">Section 3</a></li>
        </ul>

        <div class="tab-content">
          <div class="padded">
            some content goes here
            <div class="separator"></div>
            other content in the same tab pane
          </div>
        </div>
        ....
      </div>

Use class separator to break the contents in the same tab pane.
