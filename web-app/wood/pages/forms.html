Full-width inputs

If you want the inputs inside a form to span to the full width of their parent container, add the class "full-width" to your form:

  
  <form class="full-width"> ... </form>

Inputs

  
  <div class="input"><input type="text" placeholder="Email"></div>

Validations

  
    <div class="error full-width">
      <div class="input">
        <input type="text" id="inputError" class="error">
        <span class="input-error" data-title="Oops! This is an error!" data-original-title="">
          <i class="icon-exclamation-sign"></i>
        </span>
      </div>
    </div>

Checkboxes

The checkboxes and radio buttons are styled using Uniform JS.

  
    <label class="checkbox">
      <input type="checkbox" value="">
      Option 1
    </label>
    <label class="checkbox">
      <input type="checkbox" value="">
      Option 2
    </label>

For inline checkboxes, add "inline" class to your "label" tag.

Radio buttons

  
    <label class="radio">
      <input type="radio" name="optionsRadios" id="optionsRadios1" value="option1">
      Option 1
    </label>
    <label class="radio">
      <input type="radio" name="optionsRadios" id="optionsRadios2" value="option2">
      Option 2
    </label>

For inline radio buttons, add "inline" class to your "label" tag.

Selects

This theme contains tools for creating simple yet stylish select tags, as well as multiple select tags.

  
      <select class="chzn-select" data-placeholder="Choose a city...">
        <option value></option>
        <option>Amsterdam</option>
        <option>Bucharest</option>
        <option>Cairo</option>
        <option>London</option>
        <option>Paris</option>
      </select>

For multiple selects add multiple to the select tag: 

  
      <select multiple class="chzn-select" data-placeholder="Choose multiple cities...">

Search inputs

The search inputs are styled as normal inputs, appended or prepended by buttons.

Example:

  
      <form class="form-search">
        <div class="input-append">
          <input type="text" class="search-query">
          <button type="submit" class="btn">Search</button>
        </div>
      </form>

Appended and prepended inputs

$ .00 
  
      <div class="input-prepend input-append">
        <span class="add-on"> $ </span>
        <input id="appendedPrependedInput" type="text">
        <span class="add-on">.00</span>
      </div>
