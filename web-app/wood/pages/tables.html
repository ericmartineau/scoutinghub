Tables

Use a thead and a tfoot to get a table header and a table footer. They integrate very good with the content boxes.

    
    <table class="table table-bordered"> ... </table>

If you want your table to have no borders, add class noborder to it.

    
    <table class="table noborder"> ... </table>

Of course, you can add buttons, labels, badges and other elements to your table cells with no restriction.
