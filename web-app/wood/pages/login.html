Login/register page

The login/register page uses customized inputs with suggestive icons.

The syntax:

  
    <div class="login-container center">
      <div class="title">Login</div>

      <form class="login-form">
        <ul class="fields-list">
          <li>
            <i class="icon-user" style="float: left;"></i>
            <input autocomplete="off" id="username" name="username" placeholder="username" type="text">
          </li>
          <li>
            <i class="icon-key"></i>
            <input autocomplete="off" id="password" name="password" placeholder="password" type="password">
          </li>
        </ul>

        <div class="actions">
          <input class="btn btn-green btn-large login-btn" name="commit" type="submit" value="Sign In">
        </div>

        <div class="social-actions">
          <a href="#" class="btn btn-twitter btn-block">
            <i class="icon-twitter-sign"></i> Login with Twitter
          </a>
          <a href="#" class="btn btn-facebook btn-block">
            <i class="icon-facebook-sign"></i> Login with Facebook
          </a>
        </div>
      </form>
    </div>

It also provides buttons for Facebook login and Twitter login.
